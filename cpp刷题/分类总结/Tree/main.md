# 树容易出现超时的问题：

层次遍历如果计算了null节点，大概率都要超时：

1. unsigned long long来代替long long或者int.
但是这时候取最大值的时候容易出问题。

2. 使用不能计算空节点，而是让每个节点都带有自己的`index, index*2, index*2+1`，让index值相减
